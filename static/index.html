Application Overview

MirrorMirror is a smart mirror that provides users with a personalized experience. The mirror displays real-time information such as weather, news, and traffic updates, as well as the user's calendar, reminders, and other notifications. It also includes features such as voice recognition, touchless controls, and customizable profiles for multiple users.

Technology Overview

  • Libraries: jQuery, Bootstrap
